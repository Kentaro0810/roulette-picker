# roulette-picker
# ルーレット抽選アプリ

項目を入力して、ルーレットでランダムに1つ選べるWebアプリです。

## 機能

- 項目を1行ずつ入力
- 入力項目をルーレットに反映
- ルーレットアニメーション
- ランダム抽選
- 項目の保存
- 抽選結果の履歴表示
- 履歴削除
- スマホ対応

## 使い方

1. テキストエリアに項目を1行ずつ入力する
2. 「反映」を押す
3. 「ルーレットを回す」を押す
4. 結果が表示される
5. よく使う項目は「保存」で保存できる
